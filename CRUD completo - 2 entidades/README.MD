# Com base no código demonstrado em aula, do repositório: 

https://github.com/lcsgonzalez/aula_05022026_senai

Concluir a implementação das funções CRUD da entidade Aluno, lembrando de realizar todas as validações abaixo:

- O nome do aluno:
não pode conter números;
não pode conter caracteres especias;
não pode estar em branco ou somente com espaços;
não pode ser nulo.
- A dataNascimento:
precisa ser uma data válida;
o aluno precisa ter, pelo menos, 14 anos (na data do cadastro);
o aluno pode ter, no máximo, 130 anos (na data do cadastro);
não pode ser nula.
- A turma:
deve ser uma turma válida;
deve ser uma turma ativa.
