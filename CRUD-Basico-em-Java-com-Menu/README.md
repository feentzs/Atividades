# CRUD Básico em Java com Menu
Nesta atividade, você deverá desenvolver um programa em Java que implemente um CRUD básico (Create, Read, Update e Delete) utilizando um menu interativo no console, conforme os requisitos abaixo:

O sistema deverá permitir o gerenciamento de nomes, sendo esse o único dado manipulado. Os nomes devem ser armazenados em uma lista (por exemplo, ArrayList<String>).

O menu deve apresentar as seguintes opções:

1 - Cadastrar nome
2- Listar nomes cadastrados
3 - Atualizar um nome existente
4 - Remover um nome
5 - Sair do sistema

O programa deve permanecer em execução até que o usuário escolha a opção Sair.
Para as operações de atualização e remoção, você deve solicitar ao usuário o índice ou o nome que deseja alterar/remover.

O objetivo da atividade é praticar:

   - Uso de listas em Java
   - Estruturas de decisão (switch ou if)
   - Laços de repetição
   - Entrada de dados pelo Scanner
